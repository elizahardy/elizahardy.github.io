Python Turtle Project

About the Project

The project makes a scene by using Python Turtle Graphics. I used pre-loaded functions (draw_circle, draw_rectangle, draw_square, and draw_triangle) to make more complex shapes such as stars, a moon, and a castle. By rotating and shifting shapes in the same function I was able to create these shapes.

Sample Code

This portion of code makes the stars by combining two triangles that overlap eachother. The parameters used let the positioning of the stars to be flexibly placed around the scene.


• t: The object for drawing in turtle Graphics
• x, y: Gives coordinates on where to start drawing the stars
• b: Signals how many pixles to move by in a stroke
def draw_star(t, x, y, b): """Draw six pointed star""" t.pencolor("yellow") jump_to(t, x, y) draw_triangle(t,b,"yellow") jump_to[t,x+b,y+(2*b/3)] t.right(180) draw_triangle(t,b,"yellow")

Final Scene

This is the compleated scene using turtle graphics. It includes a crecent moon, six stars, and a castle with two towers on a dark blue background:

This scene uses a variety of shapes provided by turtle to make new ones. The castle uses a rectangle and squares, the six stars use two triangles each(one of which is flipped horizontially), and the moon combines two different colored circles. All elements use layering in order to depict a better story. After running the code, a scene of a castle in the night is revealed.
